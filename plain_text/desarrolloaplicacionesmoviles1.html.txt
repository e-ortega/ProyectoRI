       Home  Archivo  Contacto      Webadicto.Net                     Aprende a Programar Aplicaciones para Android  Escrito por KhronoX el  2 julio 2011 a las 12:15           Android es actualmente la plataforma con mayor cuota de mercado en el mundo ( según reportes de la IDC ) superando a Nokia, iPhone y Blackberry. Se estima que 300,000 teléfonos móviles con Android son activados diariamente. La gran demanda por dispositivos móviles que corran bajo esta plataforma, genera un gran interés por parte de los desarrolladores que desean crear aplicaciones para un mercado en pleno auge. Si eres desarrollador/a y deseas aprender a crear aplicaciones para Android, aquí encontrarás una serie de recursos que te ayudarán a iniciarte en esta gran aventura.  Para desarrollar aplicaciones para dispositivos móviles que funcionen bajo el sistema operativo Android , deberás tener nociones básicas sobre programación . Si conoces el lenguaje C o C++, entonces no tendrás muchos problemas, ya que para crear aplicaciones que funcionen en Android vas a tener que programar en Java cuya sintaxis es casi la misma. Lo que sí vas a tener que aprender es a utilizar una serie de librerías y componentes propios de la plataforma. Pero no te preocupes que todos estos detalles los encontrarás en los recursos que verás a continuación.  Guías, Tutoriales y Recursos para Aprender a Desarrollar Aplicaciones para Android  Lo primero que debes aprender como desarrollador de aplicaciones Android es el lenguaje de programación Java . No es necesario ser un experto en el manejo del lenguaje Java, pero al menos debes conocer y manejar con fluidez el lenguaje.  Aprende el Lenguaje de Programación Java   Manual de Java en pdf: Aprenda Java como si estuviera en primero . El manual es poco antiguo, pero lo importante es aprender a manejar el lenguaje Java , su sintaxis, sentencias, instrucciones básicas.
