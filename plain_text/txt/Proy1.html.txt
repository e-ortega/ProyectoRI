En él su autor usaba una Raspberry Pi conectada a un Kindle modificado para mostrar los datos de un paseo con bicicleta, tales como kilómetros recorridos, tiempo empleado, etc… Aquí tenéis el enlace directo al proyecto . Usar una Raspberry Pi 2 como media center con Kodi Si ya tienes una Raspberry conocerás los componentes y características que tiene, una salida HDMI, una conexión de red y un procesador ARM de una muy buena velocidad, así que la Raspberry Pi 2 y también la primera versión son una buena plataforma para la construcción de un centro o servidor de medios multimedia perfecto. Hay unos cuantos programas que te pueden servir para hacerlo pero en este tutorial su creador usa el Kodi (antiguo XBMC)  para realizarlo en apenas media hora. Enlace al proyecto . Usar Raspberry Pi y Siri para controlar el garaje, las luces de casa, el termostato,… En este proyecto puedes hacer que la Raspberry Pi controle un sistema domótico facilmente ya que se puede usar la Raspberry Pi conectada a un Iphone en concreto a Siri mediante el software SiriProxy para crear un comando personalizado en la propia Siri y así poder controlar todos los aparatos domésticos que puedas tener conectados a una red, por ejemplo apagar o encender las luces de tu casa, abrir la puerta del garaje,… Enlace al proyecto . Moccapi, una cafetera con Raspberry Pi Gracias al proyecto MoccaPi puedes controlar una cafetera con una Raspeberry Pi pudiéndola programar para que cuando llegues a casa te este esperando tu café favorito recién hecho. Enlace al proyecto .     Construir un iBEACON con vuestra Raspberry Pi ¿Os acordáis de la peli de Minority Report, cuando las marquesinas de anuncios reconocían al que pasaba cerca de ellas para mostrarle publicidad personalizada? Bueno pues eso existe y se denomina iBEACON , son unos sensores que sirven para posicionar un objeto en interiores. Este tutorial es para nivel avanzado y necesitas de varios dispositivos especiales que no son de encontrar en casa, pero eso si habrás creado una moderna herramienta que gracias a una aplicación para el móvil podrá detectarte y enviarte a la pantalla del móvil información personalizada. Enlace al proyecto .
